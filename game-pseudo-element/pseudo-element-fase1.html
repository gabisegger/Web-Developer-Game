::Pseudo-Elementos

Para começarmos, o pseudo-elemento é...

  • 1- uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado.
  • 2- utilizado para estilizar um elemento baseado em seu estado.
